Download laptop client

Install on your exam laptop once. Use your license URL on phone.

Linux

curl -fsSL https://raw.githubusercontent.com/linaelmarzouki-etu-dev/interview-screen/main/install-linux-client.sh | bash

Then run: ~/interview-screen-client/start-laptop-client.sh

Windows

Download and run install-windows-client.bat from GitHub, then run start-laptop-client.bat.

Download ZIP (Windows)

Phone (your license link)

http://server:8765/u/ABCDEFGH

Open it on your phone → auto login → tap Grab laptop screen.
