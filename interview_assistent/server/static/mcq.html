How to use

  • Phone (stealthiest): Photograph the exam screen, upload here — nothing shows on shared screen.

  • Grab laptop screen: Phone taps GRAB → laptop agent captures exam screen → answer here.
  • Before exam (laptop once): Run ./start_laptop_agent.sh and leave it running.
